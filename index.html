Hi, my name is James.

I'm a software engineer based in Los Angeles, CA.

I am a co-creator and software engineer at Guardrail, an open-source
traffic replay tool for testing microservices in isolation.

Learn More

Guardrail is an open-source traffic replay tool for testing microservices in isolation. It records production traffic and replays it in the staging environment to simulate what traffic the microservice needs to handle.

At a glance, it lets developers understand how changes to the microservice will affect the API.
